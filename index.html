Finding Points of Interest via Statistics

In this chapter, Cairo describes how data visualizations can be used to find interesting patterns within the data. This first involves using statistics to find trends that could prove illuminating when displayed by the proper visualization. For example, he talks about using averages (mean, median, and mode) to create a baseline that can be used to put outliers into context. Median and mode are resistant statistics, meaning that outliers have a minimal effect on them. Mean on the other hand is greatly affected by outliers, so comparing all three values can highlight outliers in a single direction. This is only one example of how statistical analysis can be used as a starting point. Other characteristics such as distribution and standard deviation can also be used in the same way.
